Apache 제품 보안 업데이트 권고

2025-03-19

□ 개요
 o 아파치 소프트웨어 재단은 자사 제품에서 발생하는 취약점을 해결한 보안 업데이트 발표 [1]

 o 영향받는 버전을 사용 중인 사용자는 해결 방안에 따라 최신 버전으로 업데이트 권고



□ 설명
 o Apache Tomcat에서 발생하는 원격 코드 실행 취약점(CVE-2025-24813) [1][2]
